### Requirements

- Node.js 16.19.1
- Npm 8.19.3
- MySQL 8.0.31

### Installation

```bash
git clone https://github.com/mangojang/react_twitter_clone.git 
cd ./react_twitter_clone
```

### Frontend

- .env setting
    
    ```json
    COOKIE_SECRET= Your cookie secret
    NODE_ENV=dev
    ```
    

```bash
cd ./front
npm install
npm run dev
```

### Backend

- .env setting
    
    ```json
    COOKIE_SECRET= Your cookie secret
    DB_PASSWORD = Your db password
    FRONT_URL= Your front url
    ```
    

```bash
cd ./back
npm install
npm run dev
```
